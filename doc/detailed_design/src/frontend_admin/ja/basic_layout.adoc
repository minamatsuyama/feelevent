* ヘッダー
** タグ：header

ヘッダー内にはメニューのナビゲーションが入る。

* コンテンツ
** タグ：section
** class：content

コンテンツ要素が入る。

* フッター
** タグ：footer
** id：global-footer

コピーライト等の付加情報が入る
